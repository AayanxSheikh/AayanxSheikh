# Aayan Sheikh

Software engineering intern working on AI tooling for healthcare and small-business web infrastructure.

---

### Currently working on

- **[BillCheck AI](https://github.com/AayanxSheikh/billcheck-ai)** — AI-powered medical bill validator. FastAPI backend over a PostgreSQL database of 7,211+ CPT codes from CMS sources, with a Tesseract OCR pipeline that flags 150–300% markups against Medicare pricing. Next.js frontend.
- **Backend Developer Intern @ Scotts Valley Chamber of Commerce** — Rebuilt the chamber site for performance and SEO: PageSpeed mobile usability 64 → 91, page load time down 40%, schema markup across 50+ pages.
- **Web Development Intern @ Think Local First** — Python scraper for 800+ listings feeding a Levenshtein fuzzy matcher (null-result searches down 52%); composite SQL indexes cut query latency from 850ms to 210ms; Lighthouse 72 → 93.
